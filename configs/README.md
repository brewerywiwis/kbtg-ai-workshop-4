# Configuration

This directory contains configuration templates and default configurations for the LBK Points Transfer application.

## Files

- `app.yaml` - Main application configuration
- `database.yaml` - Database configuration

## Environment Variables

The application supports the following environment variables:

- `PORT` - Server port (default: 3000)
- `DATABASE_URL` - SQLite database file path (default: users.db)
- `LOG_LEVEL` - Logging level (default: info)
